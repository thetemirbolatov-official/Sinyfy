# Changelog

## [1.0.0] - 2024-01-15

### Added
- Первый публичный релиз
- Базовая функциональность клонирования сайтов
- Извлечение CSS из тегов style и внешних файлов
- Преобразование инлайн стилей в классы
- Скачивание изображений с созданием заглушек
- Поддержка data URI
- Обработка фоновых изображений в CSS
- Удаление JavaScript и обработчиков событий
- Создание манифеста с информацией о ресурсах
- Поддержка аргументов командной строки
- Логирование операций

### Features
- Полная автономная копия сайта
- Без использования Selenium
- Кроссплатформенность
- Простой API для использования как библиотеки

### Technical
- Поддержка Python 3.7+
- Использование requests для HTTP запросов
- Использование BeautifulSoup4 для парсинга
- Использование Pillow для обработки изображений
